WDU Inquiry Form will replce default Add to cart button

== Description ==

Inquiry Form for Woocommerce just replace your add-to-cart button by a custom button which will bring a popup with a form. The data of the form will go to the admin email.
If you need to put a quotation or inquiry form instead of add-to-cart in Woocommerce single product page then this plugin is fantastic.

Demo:

**[LIVE DEMO](http://akramul.com/plug-dev/shop/)**

= Features =
* Simple & Intuitive
* Option for using built-in form or custom form
* Bootstarap switcher
* Very easy to use

== Installation ==

Upload the WDU Inquiry Form plugin to your Woocommerce site, Activate it, and enjoy.

== Frequently Asked Questions ==

= What does actually do it? =
This plugin simply replaces your Add to cart button by a custom button which will bring up a popup form.

= Is this plugin remove add to cart from Shop/Archive page? =
Yes, it removes all “Add to cart” from every place.

= Is it will add the inquiry form in bottom tab area? =
No.
It will replace default add to cart button and put a new button. After hitting this button, a form will come with popup.

= Can I use my custom form? =
Yes.
